📋 技术原型说明

本Demo展示了能源统计系统中使用的所有前端技术（除SQL外）：

  • rowspan / colspan - 单元格上下合并 / 左右合并
  • data-month - 自定义数据属性，用于列高亮和点击交互
  • addEventListener - 鼠标悬停、点击事件监听
  • 动态表格生成 - JavaScript动态创建HTML表格元素
  • 空表格初始化 - 页面加载时先显示表格结构
  • 自动触发查询 - 页面加载后自动填充数据
  • CSS Hover效果 - 行hover、列hover（月份列）

💡 提示：打开浏览器控制台(F12)查看详细的技术说明和执行日志

🔧 关键代码片段

1. 上下合并（rowspan）：

tdPlace.rowSpan = 3; // 合并3行

2. 左右合并（colspan）：

th.colSpan = 2; // 合并2列（电量+占比）

3. 自定义数据属性：

td.dataset.month = m; // 设置 data-month="1"

4. 列高亮交互：

table.querySelectorAll(`[data-month="${month}"]`).forEach(td => td.classList.add('col-hover'));

5. 百分比计算：

(Math.round(value * 10000 / total) / 100) + "%"
